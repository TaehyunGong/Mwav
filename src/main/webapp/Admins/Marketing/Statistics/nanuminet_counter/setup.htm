접속통계 소스 자동 셋업(나누미넷)

※ 이 프로그램은 모두 자유롭게 이용 하실 수 있습니다.
    해당 프로그램은 Microsoft SQL2000에서 사용가능하고 데이터 베이스연결은 OLEDB방식을 사용하였습니다.

 
※ 주의
     하부 디렉토리 static 폴더는 필히 쓰기권한이 있어야 합니다.
 
 타이틀(적용될사이트이름)은 사이트명을 입력하시면 됩니다.
 네임(Table이름)은 생성되는 테이블명입니다.가급적 영문을 기입하십시요.
 DB서버이름(컴퓨터아이디나 아이피주소)에는 해당 DB의 서버의 IP주소 ,도메인이나 호스트명을 입력하십시요.
    예> db.자기도메인 
 DB이름(해당 DB) ftp_id입니다.
 DB로그인유저(해당 DB유저계정) 는 ftp_id입니다..
 DB로그인비밀번호(해당유저 password) 는 ftp_pwd입니다.
타이틀(적용될사이트이름)  
네임(Table이름)  
DB서버이름( db.자기도메인)  
DB이름( ftp_id)  
DB로그인유저( ftp_id)
DB로그인비밀번호( ftp_pwd)
